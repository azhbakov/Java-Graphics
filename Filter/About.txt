Класс Logic - модель, содержит оригинальные изображения для зон, параметры представления. AppWindow - view, содержит Zone's - панели, представляющие изображения из модели.
Класс PaintArea - панель, с перегруженным методом paint() для рисования рамки при выделении области пользователем.
Пакет BMP содержит структуры и классы для работы с BMP изображениями.

Статические методы для преобразования изображений содержатся каждый в своем классе в пакете model.
